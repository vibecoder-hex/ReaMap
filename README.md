# ReaMap
A. Проектирование базы данных(ТВОЯ ЗАДАЧА).
1. Придумать идею задачи проектирования базы данных.
2. Выявить и описать основные сущности из задачи.
3. Построить концептуальную модель и EER-диаграмму(лучше в воркбенче, думаю с ним работать ты умеешь)
4. Заполнить тестовыми данными сущности и вывести их(SELECT INSERT), но эту скучную задачу нужно делегировать deepseek
5. Предоставить концептуальную модель, ну и наборы SQL скриптов.
6. Описать на prisma сущности (СОВМЕСТНАЯ РАБОТА)

Б. Проектирование архитектуры приложения(МОЯ ЗАДАЧА)
1. Подготовить среду разработки: git, docker, typescript, npm, node js, express, prisma
2. Настроить проект, разместив в docker-compose сервисы: express, mysql, nginx(опционально)
3. Спроектировать простую архитектуру для кодовой базы
4. Реализовать базовые api эндпоинты(пока будет Hello world в формате json)
5. Описать на prisma сущности, настроить базу данных в backend-приложении(СОВМЕСТНАЯ РАБОТА)
6. Реализовать JWT-аутентификацию(на массивах пока-что)
7. Реализовать CRUD операции в приложении.
8. Организовать I/O данных в формате JSON.

В. ОПЦИОНАЛЬНО. Разработка интерфейса.
1. Спроектировать базовый UI/UX.
2. Разработать простой интерфейс на базе Vue.js компонентов с использованием готовых CSS стилей(bootstrap/vuetify)
